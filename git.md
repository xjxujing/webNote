安装git后 git --vesion 查看版本

~~~shell
# 初始化本地git仓库
git init 

# 添加文件
git add <file> 

# 查看状态
git status

# 提交到本地仓库
git commit

# 推送到远程仓库
git push

# 远程拉取数据
git pull

# 远程拷贝仓库
git clone 
~~~
